# High Ground Odyssey — W06 Project Example (Brand-aligned)

This is a complete example site built to satisfy the WDD 131 Week 06 project rubric *and* align with the High Ground Odyssey brand direction.

## Pages
- index.html — home (latest episode + preview + visits via localStorage)
- episodes.html — browse + filter + save/remove episodes (localStorage)
- contact.html — standards-based form with JS submit handling and localStorage name-prefill

## Rubric features hit
- Valid, semantic HTML; external CSS and JS
- Responsive design principles: proximity, alignment, repetition, contrast, whitespace
- JavaScript:
  - multiple functions, DOM manipulation, event listeners
  - conditional branching
  - objects + arrays + array methods
  - template literals for output
  - localStorage (visits, saved episodes, preferred name)
- Lazy loading: images below the fold include loading="lazy"

## Assets
Uses Journey.jpg + WebP derivatives and Valknut logo SVGs in /images.
